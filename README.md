# Student-Staff-Manager

## Tech Stack

- **Backend**: Laravel 12.0, PHP8.2
- **Frontend**: Bootstrap 5.1.3
- **Database**: MySQL
- **Package Management**: Composer
- **Excel Processing**: Maatwebsite Excel 3.1
- **Data Tables**: Yajra DataTables 12.7
- **Authentication**: Laravel Breeze 2.4
- **Icons**: Font Awesome 6.0
- **JavaScript**: jQuery 3.6.0
